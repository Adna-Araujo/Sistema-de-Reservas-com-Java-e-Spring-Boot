# Sistema-de-Reservas-com-Java-e-Spring-Boot
# 🗓️ Sistema de Reservas de Salas - API REST

API desenvolvida para gerenciar reservas de salas de reunião, focada em regras de negócio de disponibilidade e integridade de dados.

## 🛠️ Tecnologias Utilizadas
- **Java 21**
- **Spring Boot 3**
- **Spring Data JPA** (Persistência)
- **PostgreSQL** (Banco de dados)
- **Maven** (Gerenciador de dependências)
- **Bean Validation** (Validação de dados)

## 🚀 Funcionalidades
- [ ] Cadastro e gestão de salas.
- [ ] Sistema de reservas com validação de horários.
- [ ] Tratamento global de erros para respostas amigáveis em JSON.
- [ ] Persistência de dados segura com Spring Data JPA.

## ⚙️ Como executar
1. Clone o repositório.
2. Configure o banco de dados no `application.properties`.
3. Execute o comando: `./mvnw spring-boot:run`
